# ServiceNow Project

## Project Name
Metro Ticket Booking System

## Description
This project was developed using the ServiceNow platform to manage metro ticket booking and related services.

## Features
- User Registration
- Ticket Booking
- Ticket Cancellation
- Booking Status
- Email Notifications

## Technologies Used
- ServiceNow
- Flow Designer
- Tables
- Forms
- Notifications
